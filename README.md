# Churn_Analysis

This project is a Customer Churn Analysis using Power BI, designed to help businesses understand why customers leave and identify key factors influencing retention.

Overview

Built an interactive dashboard to visualize churn trends, customer behavior, and business performance.

Used Power Query for data cleaning and transformation.

Created DAX measures for calculating key metrics like churn rate, retention rate, and segment-wise analysis.

Enabled stakeholders to explore dynamic insights through slicers and visualizations.

Features

Churn rate analysis by customer segment

Visual representation of retention patterns and trends

Identification of high-risk customer groups

Interactive filtering for detailed insights

Tools & Technologies

Power BI – Dashboard and visualization

Power Query – Data cleaning and transformation

DAX – Measures and calculations

Excel / CSV – Source data

Outcome

Provides actionable insights for improving customer retention

Helps in strategic decision-making and understanding customer behavior
